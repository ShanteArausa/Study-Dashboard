# Study Dashboard

A JavaScript-based study tracker designed to help learners log, track, and analyze their study sessions.

## Features

- Log study hours per session
- See total hours and average study time
- Delete individual sessions
- Dynamic visual feedback (average hours color & progress bar)
- Persistent data with localStorage

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Browser localStorage
